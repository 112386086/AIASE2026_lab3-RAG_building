The D-Bus interface specification for 'linux-devicetree/Documentation/devicetree/bindings/gpio/mediatek,mt7621-gpio.yaml' is defined as follows:
Description: The IP core used inside these SoCs has 3 banks of 32 GPIOs each.
The registers of all the banks are interwoven inside one single IO range.
We load one GPIO controller instance per bank. Also the GPIO controller can receive
interrupts on any of the GPIOs, either edge or level. It then interrupts the CPU
using GIC INT12.
